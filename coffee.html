200° Coffee
— Digital Design Suite

Interactive PDF, EPUB, social carousel, poster, and data visualisation — a full digital design suite for one of the UK’s leading independent coffee roasters.

Overview

A full digital suite for a premium coffee brand

200° Coffee is one of the UK’s most recognisable independent coffee roasters — a brand known for quality, character, and a strong visual identity across its Nottingham-based stores and roastery. The brief covered multiple formats: an interactive PDF and EPUB guide, a social media carousel, a poster, and a data visualisation piece presenting customer satisfaction survey results.

My role as sole designer was to take the brand’s existing visual language and apply it coherently across five distinct digital formats — each with different layout constraints, audience contexts, and technical requirements.

Challenge vs. Response
Challenge

Five separate outputs — PDF, EPUB, carousel, poster, data vis — each requiring a different design approach, while maintaining a coherent visual identity across all of them.

Response

Established a consistent design system first — red, white, and dark palette, Helvetica Neue type hierarchy, and a set of reusable layout principles — then adapted it to the specific constraints of each format.

Challenge

The interactive PDF and EPUB needed to guide users through a six-step cappuccino-making process in a way that was clear, engaging, and navigable across devices.

Response

Designed an icon-led step system with a persistent navigation bar (Intro / Guide / Survey / About Us) giving users clear orientation at every point.

Challenge

Customer satisfaction survey data needed to be presented in a way that was honest, readable, and on-brand rather than feeling like a generic chart.

Response

Designed a custom data visualisation using the brand palette — leading with the positive headline (66% satisfied) while presenting all five segments clearly as a design element, not an afterthought.

Five formats, one brand. The discipline was making each deliverable feel native to its medium without losing the thread back to 200° Coffee’s visual identity.

— 200° Coffee, 2024
Process

From brief to five outputs

01
Brand immersion + design system
Studied 200° Coffee’s existing visual identity. Established the core design system — colour, type scale, iconography, and layout grid — so all five outputs could be built from the same foundation.
02
Interactive PDF + EPUB
Designed and built the interactive publication in InDesign — structured navigation, step-by-step icon guide, survey data section, and About Us page. Exported to both interactive PDF and EPUB, ensuring both formats functioned correctly across different environments.
03
Data visualisation
Took raw customer satisfaction survey results — five response segments — and designed a clean, brand-consistent visualisation. Structured to lead with the positive headline (66% satisfied) while giving full visibility to all response categories.
04
Poster design
Designed a print-ready poster using the brand’s coffee imagery and typographic system — bold headline, structured body copy, and a strong visual hierarchy built around contrast and whitespace.
05
Social media carousel
Adapted the cappuccino guide into a social-native carousel format — one step per slide, each frame designed to stand alone while reading as a sequence. Optimised for mobile scroll behaviour.
